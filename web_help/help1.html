Работа с базой данных


Для перехода по записям в базе данных используйте колесо мыши либо нажимайте на зеленые стрелки в верхней части окна.
Для добавления записи нажмите на "+". Затем нажмите на галочку для подтверждения добавления и затем на кружок из двух стрелок для обновления таблицы.
Для редактирования нажмите на значок карандаша. Затем нажмите на галочку для подтверждения редактирования и затем на кружок из двух стрелок для обновления таблицы.
Для удаления нажмите на значок корзины.

Настройки интерфейса


Вы можете настроить шрифт, изменяея размер, начертание, стиль и цвет, кажав на кнопку "шрифт", цвет фона формы, нажав на кнопку "цвет" или установить цвет по умолчанию.
Кнопка "Установить фононвое изображение" устанавливает картинку как фон. Картинка только в формате PNG.
Кнопка "Фон по умолчанию" убирает картинку и возвращает цвет фона, который был до этого.
Можно выбрать язык кнопок. Для того, чтобы переключился язык необходимо перезапустить програму.

Категории пользователей

Всего есть 3 категории пользователей.
1. Администратор. Администратору доступны все категории действий, должен регистрировать новых пользователей и наделять их правами доступа.
2. Оператор. Оператор может редактировать данные в представлениях, может создавать новые таблицы.
3. Пользователь. Пользователь может только просматривать данные.
